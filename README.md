# 🚀 JavaScript Projects Collection

This repository contains a collection of beginner-to-intermediate JavaScript projects built while learning modern JavaScript. Each project focuses on different concepts such as DOM manipulation, Local Storage, Fetch API, Event Handling, Array Methods, and responsive UI development.

---

# 📂 Projects Included

## 📝 01. Todo App with Local Storage

A simple and interactive Todo application that allows users to manage their daily tasks.

### ✨ Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* 💾 Data persistence using Local Storage
* 📱 Responsive design

### 📚 Concepts Used

* DOM Manipulation
* Event Listeners
* Local Storage
* Arrays & Objects

# 🎯 Purpose

# 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/omkarkhopade/Projects_in_JS.git
```

### 2️⃣ Navigate to any project folder

```bash
cd Projects_in_JS/01_todo-localstorage
```

### 3️⃣ Open the project

Open the `index.html` file in your browser or use the **Live Server** extension in VS Code.
