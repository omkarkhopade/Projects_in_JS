# 🚀 JavaScript Projects Collection

This repository contains a collection of beginner-to-intermediate JavaScript projects built while learning modern JavaScript. Each project focuses on different concepts such as DOM manipulation, Local Storage, Fetch API, Event Handling, Array Methods, and responsive UI development.

---

## 📂 Projects Included

### 📝 01. Todo App with Local Storage
A simple and interactive Todo application that allows users to manage their daily tasks.

#### Features
- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 💾 Data persistence using Local Storage
- 📱 Responsive design

**Concepts Used**
- DOM Manipulation
- Event Listeners
- Local Storage
- Arrays & Objects

---

### 🌤️ 02. Weather App
A weather application that fetches real-time weather information using the OpenWeather API.

#### Features
- 🌍 Search weather by city
- 🌡️ Display temperature
- ☁️ Weather description
- ❌ Error handling for invalid city names

**Concepts Used**
- Fetch API
- Async/Await
- JSON
- API Integration
- Error Handling

---

### 🛒 03. E-Commerce Shopping Cart

#### Features
- 🛍️ Display products

- ❌ Remove products from cart
- 💰 Calculate total price
- 💾 Cart persistence using Local Storage

**Concepts Used**
- DOM Manipulation
- Local Storage
- Arrays

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/<your-username>/Projects_in_JS.git
```

2. Navigate to any project folder

```bash
cd Projects_in_JS/01_todo-localstorage
```

3. Open `index.html` in your browser or use **Live Server** in VS Code.
